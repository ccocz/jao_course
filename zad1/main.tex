\section*{Solution}
Let's assume that such language $L$ exists. To check if $L$ satisfies Pumping lemma, let's apply pumping to any word $w$ = $xyz\in L$  with length $\ge$ $N$, where $N$ is pumping length. Let $a$ and $b$ be the number of a's and b's in $w$ respectively, therefore \(a = b^2\). Similarly let $a'$ and $b'$ be number of a's and b's in the subword $y$ respectively. After pumping $a + 1$ times the word $w$, the word $xy^{a+1}z$ with $a + aa'$ a's and $b + ab'$ b's should satisfy following:
\[a + aa' = (b + ab')^2\]
\[a + aa' = b^2 + 2bab' + (ab')^2\]
\[aa' = 2bab' + (ab')^2\]
\[a' = 2bb' + ab'^2\]
which is correct only when \( a' = b' = 0 \) because \(a \ge a'\),
but that would imply the word $y$ is empty which is not, hence contradiction.
\qed
